This video shows how to use a new function in 2013 - NUMBERVALUE. It converts text to a number. In some instances you may have currency numbers that are recognized as text in Excel. You need to convert this to a currency number that is recognized as a number (that Excel can perform calculations on) so the NUMBERVALUE function can do this.
