Explore the changes that have taken place as the automotive industry-renowned risk management assessment tool ‘MMOG/LE’ has evolved from previous version 4.1 to version 5 (2020). AIAG and ODETTE co-present this 40 minute webinar that covers the changes to the MMOG/LE scoring framework and concept enhancements, as well as provides a detailed overview to help with your initial set up in the new, cloud-based environment (“NP” – New Platform). Guillaume Francois and Beysun Sen Duru of the European-based ODETTE organization join James Liegghio, AIAG’s Supply Chain Program Manager, to provide this information to the global user community.
