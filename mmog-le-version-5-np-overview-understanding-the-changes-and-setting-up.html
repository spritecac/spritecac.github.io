Explore the changes that have taken place as the automotive industry-renowned risk management assessment tool ‘MMOG/LE’ has evolved from previous version 4.1 to version 5 (2020). Guillaume Francois and Beysun Sen Duru of the European-based ODETTE organization join James Liegghio, AIAG’s Supply Chain Program Manager, to provide this information to the global user community.
