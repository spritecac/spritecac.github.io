Derek Martin joins Scott Hanselman to discuss the Azure Kubernetes Service (AKS) Secure Baseline reference implementation from the Azure Architecture Center. Learn what it is, how it works, and how you can use it to start your production deployment in a secure and compliant manner. You’ll also learn about the available additions for Payment Card Industry Data Security Standard (PCI-DSS 3.2.1) compliance and multi-region cluster support.
