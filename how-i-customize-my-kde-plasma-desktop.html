For the icons, I use the "Inverse-blue-dark" icons, which you can get from the "Get new icons" button.

The window decorations I use are called Layan. You can get them from the "‚ÄòGet New Window decorations".

The widgets I use are called "Window title applet", "Latte separator", "Better Inline clock", and "latte spacer".

I have a top bar with a global menu, and a dock to run and manage running apps. I don‚Äôt use the regular Plasma panels to get that result, though, I use latte dock.

- Right click on the dock, and select "edit dock", then right click again and select "Ad dock / panel", then "Empty panel"

- Finally, there‚Äôs another latte spacer and finally, the system tray widget from the base Plasma desktop.
