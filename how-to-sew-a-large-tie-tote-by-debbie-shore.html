This large tote bag is perfect for work, weekends away or for storage!
Materials are from www.debbieshoresewing.com



Materials:
59” x 19.5” [150cm x 50cm] outer fabric
59” x 19.5” [150cm x 50cm] lining fabric
32” x 19” x [81cm x 48cm] fusible fleece [H640]
4” x 22” [10cm x 56cm] interfacing for the handles
Round template such as a saucer
Magnetic snap


Cut:
Two pieces of outer and two of lining fabric each measuring 14” x 16” [36cm x 41cm]
Two pieces of outer fabric each measuring 4” x 22” [10cm x 56cm] for the handles
Four pieces of outer fabric each measuring 3” x 6” [7.5cm x 15cm] for the tabs
Two pieces of lining each measuring 4.5” x 24” [12cm x 62cm] joined together for the tie

Fuse fleece to the wrong sides of the main outer pieces
Fuse interfacing to the wrong sides of the handles
