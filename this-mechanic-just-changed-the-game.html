The wall dash is a mechanic that has become more and more popular particularly in the past few months. A few select pros are already beginning to implement it in the game. However, there’s a bit more to it that almost no one (not even the pros) actually knows about. This could potentially cause a massive shift in meta for certain aspects of the game.

Just when we think everything in Rocket League has been discovered, new mechanics and ideas somehow still tend to pop up. I’m finding myself always being impressed by how Rocket League keeps evolving. This mechanic, the wall dash, only started being used in pro Rocket League within the past few months. I watched a lot of RLCS X last season and never saw the wall dash used once. Now, after a slight bit of research and only a little bit of watching, I see it being used all the time by select pro players. And this new mechanic in particular that has yet to be used at the highest level still hasn’t hit that audience yet. Maybe after this video, we’ll see the few professional keyboard and mouse players start using this in almost every game. Or on the other hand, the Rocket League developers, Psyonix, might find a way to patch it and make it so no one is capable of pulling this off that easily.
