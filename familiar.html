이번 강의에서는 [ 낯이 익다 / 낯설다 / 익숙하다 ] 등의 표현들을 활용한 다양한 영어 회화 패턴을 가지고 연습하겠습니다 :

1. 낯이 익다 + 어디서 본 것 같다.
누구/무엇 looks(sounds) familiar + I think I’ve seen(heard) 누구/무엇 somewhere.

2. 장소가 낯이 익다 + 와본 적이 있는 것 같다.
어떤 장소가 feels familiar + I think I’ve been [ there/here ] before.

3. 경험이 있어서 + (아주) 잘 안다
ex) I grew up there. So, I’m very familiar with that area.
ex) I’ve used it for a very long time. So, I’m very familiar with it.

4. 경험이 부족해서 + 아직 낯설다 / 잘 모른다
ex) I just moved here. So, I’m not very familiar with this area yet.
ex) I just bought it. So, I’m not very familiar with it yet.
