This is a guide for the Skier Task, Informed Means Armed on Escape From Tarkov

Task Objectives:
Install a WIFI camera to watch the pier
Install the WIFI camera to watch the road to port
Install the WIFI camera to watch Kiba Arms store

21,000 EXP
